# Chip8 Emulator (Rust)

Эмулятор легендарной 8-битной системы Chip8, написанный на языке Rust.  
Проект создан в учебных целях для изучения архитектуры эмуляторов, работы с памятью и графикой.

## Возможности

- Полная эмуляция процессора Chip8 (все 35 инструкций).
- Эмуляция дисплея 64x32 пикселя (монохром).
- Эмуляция 16-клавишной клавиатуры (клавиши 0–F).
- Таймеры задержки и звука (звук пока заглушен).
- Поддержка загрузки ROM-файлов из командной строки.
- Модульные тесты для проверки корректности инструкций.

## Сборка и установка

Для сборки проекта необходим установленный **Rust** и **Cargo**.  
Если Rust ещё не установлен, воспользуйтесь инструкцией на [официальном сайте](https://www.rust-lang.org/tools/install).

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ваш-логин/chip8.git
cd chip8
```
2. Соберите проект в режиме release:
```bash
cargo build --release
```

Исполняемый файл будет находиться в папке target/release/ (название зависит от настроек Cargo.toml; по умолчанию – имя пакета).

## Использование

Запустите эмулятор, передав ему путь к ROM-файлу Chip8:

```bash
cargo run -- <путь_к_rom>
```

Например:

```bash
cargo run -- demos/IBM.ch8
```

## Управление

Клавиши компьютера сопоставлены с оригинальной клавиатурой Chip8 следующим образом:
| Chip8   | Клавиша PC |
|-------  |------------|
| 1 2 3 C | 1 2 3 4    |
| 4 5 6 D | Q W E R    |
| 7 8 9 E | A S D F    |
| A 0 B F | Z X C V    |

## Тестирование

Для запуска модульных тестов выполните:

```bash
cargo test
```

Тесты покрывают основные инструкции и логику работы с памятью.
